  • Just take a look at what we've already done in our endeavour to make the perfect web development IDE.

      • Any HTML5+CSS+JS Application

        Web front-end, web site, mobile web app, code library - any type of web application can be built with Crafted. We will be adding HAML, SCSS, SASS etc. before too long.

      • Beautiful Code Editor

        Like many others, we are eternally grateful to the ACE project for the best web-based code editor in existence.

      • Develop AND Run

        Crafted is not just an IDE. It also lets you run and deploy your applications on the Crafted platform. You can manage multiple environments such as development, staging, and production.

      • Collaboration

        Crafted allows any number of users to collaborate in real-time projects, in a Google Docs manner. Git integration is right around the corner to complete the picture.

      • Fork a Project

        Fork any public or anonymous project into your own user account or anonymously.

      • Teams

        You can group Crafted users into teams, who can then be assigned read/write/admin access. Soon, teams will be a part of the special Organization account.

      • Import from Git and Zip

        Import from a Git repo or a Zip file.

      • Anonymous Mode

        You can access the Crafted IDE without even signing up. By bookmarking the URL any project you create will be only editable by you. Others can look at your code and even fork it.

      • Code Completion

        Yup - code completion. As you type, code suggestions appear below your code.

      • Private / Public Projects

        You can have as many public projects as you like - all for free. For those who want to protect their project we will soon be adding private projects.
