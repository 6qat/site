  • Just take a look at what we've already done in our endeavour to make the perfect web development IDE.

      • Any Web Application

        Any HTML5+CSS+JS front-end, web site, mobile web app or code library works in our IDE. And we're adding HAML, SCSS, SASS etc. before too long.

      • Beautiful Code Editor

        Like many others, we're eternally grateful to the ACE project for the best web-based code editor in existence.

      • Develop, Run, Host

        Crafted is not just an IDE. It also lets you run and deploy your applications on the Crafted platform. Manage multiple environments such as development, staging, and production.

      • 100% Cloud-Based & Fast

        Everything runs in your browser and is blazingly fast. You can access and fork projects without even needing an account.

      • Highly Collaborative

        Crafted allows any number of users to collaborate in real-time projects, a la Google Doc. Full Git integration is right around the corner to complete the picture.

      • Fork any Project

        Fork any public or anonymous project into your own user account or without even going to the trouble of creating an account.

      • Teams & Organizations

        You can group Crafted users into teams, who can then be assigned read/write/admin access. Soon, teams will be a part of the special Organization account.

      • Import from Git and Zip

        Import from a Git repo or a Zip file and your project can be run instantly within the IDE. Just press the Run button.

      • Anonymous Mode

        You can access the Crafted IDE without even signing up. By bookmarking the URL any project you create will be only editable by you. Others can look at your code and even fork it.

      • Code Completion

        Not much to say here. As you type, code suggestions appear below your code.

      • Private & Public Projects

        You can have as many public projects as you like - all for free. For those who want to protect their project we will soon be adding private projects.

  • These are just some of the features that we are working hard to make available in Crafted.

      • Multi-Pane Code Editing

        You can have up to 3 panes for side-by-side editing of any of your project files.

      • Instant Preview

        This is a great click-saving feature

      • Server-Side Javascript

        For those who want secure server-side javascript execution but with complete ease-of-use. Think of it as syncronous NodeJS.

      • Project Starring

        Just like you get on Github, it let's you flag other people's public projects so you can come back to them quickly and easily.

      • Version Control

        Crafted's version control leverages your Github account. You press a button in the IDE and a version gets created.

      • Resource Browser

        This let's you explore and manage your project files and is especially useful for images with multi-file upload and other cool features.

      • External Deployment

        We are adding in single button press deployment to multiple Crafted.io production locations as well as FTP and SFTP. Great for Development, Staging and Production scenarios.

      • Mobile Build

        If you're familiar with Phonegap:Build, then this is our own equivalent for converting HTML5 mobile apps into Appstore ready binaries. Comes with Certificate management, too.

      • Team Accounts

        Create Organizations and Teams for assigning Read/Write/Admin permissions as well as single-point billing.
